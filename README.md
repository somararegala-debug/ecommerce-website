# MarketLane Ecommerce

A small full-stack ecommerce demo with an HTML storefront, responsive CSS, browser cart, and Node.js backend APIs.

## Run it

```bash
npm start
```

Open `http://localhost:3000`.

## What is included

- `public/index.html` is the main HTML storefront.
- `public/styles.css` contains the responsive ecommerce design.
- `public/app.js` loads products, handles the cart, and submits checkout.
- `server.js` serves the site and exposes `/api/products` and `/api/orders`.
- `data/orders.json` stores submitted orders locally.

No external packages are required.
